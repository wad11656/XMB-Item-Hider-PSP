# XMB Item Hider PSP

XMB Item Hider plugin source for PSP. Upgraded to include option to _completely_ hide categories (not just all the items within categories).

Continuation of Frostegater's project: https://www.gamebrew.org/wiki/XMB_Item_Hider_PSP

## Build

Build with the included `Makefile` in a PSP toolchain environment:

```sh
make
```
